
# GitHub User Finder

A simple application to search and display GitHub user profiles and repositories.

## Features

- Search for GitHub users by username
- View user profile information (avatar, bio, followers, etc.)
- Display user's public repositories
- Responsive and user-friendly interface

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/GitHubUserFinder.git
cd GitHubUserFinder
```

3. Configure your GitHub API key (To avoid rate limits):
    - Create a `config.js` file in the project root
    - Add your GitHub personal access token:
    ```javascript
    export const GITHUB_TOKEN = 'your_github_token_here';
    ```
    - Generate a token at https://github.com/settings/tokens

## Usage

1. Enter a GitHub username in the search bar

2. View the user's profile information
